# Review

강의를 통해서 얻은 교훈은 다음과 같다.

1. 상상속, 미래의 문제는 선택하지 말자. (~가 걱정인 사람 등) -> 현재 문제를 찾아야 함

2. 고통은 그냥 고민이 아니라, 문제를 해결하기 위해서 많은 돈(100만원 이상)을 기꺼이 낼 것인가이다.

"누구의"는 매니아/전문가를 찾아보자. 돈을 많이 낼 사람은 매니아임. 

-> ~가 없어서로 가면 고통이 커지지 않음
(e.g. context: 차량에 이상 증상에 의해 정비소를 갔지만 여러 요소들을 교체해야한다함. problem: 전문지식이 없어서 정비소의 진단과 견적을 믿을 수 없는 운전자)

-> ~가 있음에 도 불구하고로 가면 고통이 커짐
(e.g. context: 교통사고가 나서 여러 정비소에서 확인해본 결과 신차 보증기관이 끝나 1000만원의 견적이 나옴. problem: 정비 가격이 진짜인지 확인하고 싶은 운전자)
